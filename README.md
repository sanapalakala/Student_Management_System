# Student Management System (Java + React)

A full-stack Student Management System built using Java for backend logic and React for frontend UI. The system allows users to manage student records with CRUD operations.

---

### Backend (Java)
- Add Student
- View All Students
- Delete Student
- Update Student Marks
- Search Student by ID
- Input Validation
- Duplicate ID Prevention
- OOP-based Layered Architecture

### Frontend (React)
- Controlled Form Inputs
- useState for State Management
- Add Student
- Display Student List
- Delete Student
- Dynamic Rendering using map()
